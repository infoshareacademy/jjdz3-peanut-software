PEANUT MEDICINE REPORTS API

Current version: 1.0

Peanut Medicine Reports

Get reports via a RESTful API

API ENDPOINTS

Below are described the REST endpoints available that you can use to search for reports

ALL SURVEYS

List of all registered surveys

Content-Type: application/json


{hostname}/peanut-api/reports/survey

SELECTED SURVEY

Returns selected survey by id

Content-Type: application/json


{hostname}/peanut-api/reports/survey/{id}

REPORT OF ALL PREFERRED SPECIALIZATIONS

List of all preferred specializations

Content-Type: application/json


{hostname}/peanut-api/reports/specialization

SELECTED SPECIALIZATIONS

Show number of visit for selected specialization

Content-Type: application/json


{hostname}/peanut-api/reports/specialization/{name}

USERS ACTIVITIES

Shows users activities

User email, login date-time, logout date-time

Content-Type: application/json


{hostname}/peanut-api/reports/usersActivity

ICD-9 CLASSIFICATION

Shows International Statistical Classification of Diseases and Related Health Problems

Content-Type: application/json


{hostname}/peanut-api/reports/classification
